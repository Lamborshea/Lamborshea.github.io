总结

JavaScript 中存在一种声明提升的机制，有变量声明和函数声明。JS 引擎会在编译阶段查找每个作用域的声明，而赋值和运算则在执行时进行。

函数定义分为函数声明和函数表达式，其中，函数声明有声明提升，函数表达式则没有。
